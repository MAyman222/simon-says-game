# Simon Game

A browser-based memory game inspired by the classic electronic game Simon.  
The player must remember and repeat an increasingly long sequence of colors.

This project was built using **HTML, CSS, and JavaScript** while learning DOM manipulation and event-driven programming.

---

## Demo

Press any key to start the game and try to repeat the color pattern correctly as it grows each level.

---

## Technologies Used

- **HTML5** – page structure  
- **CSS3** – styling and animations  
- **JavaScript** – game logic and interactions  
- **jQuery** – DOM manipulation and event handling  

---

## How the Game Works

1. Press any key to start the game.
2. The game will flash a random color.
3. The player must click the correct color button.
4. Each level adds a new color to the sequence.
5. The player must repeat the full sequence correctly.
6. If the player clicks the wrong button, the game ends and can be restarted.

---

## Features

- Interactive button animations
- Sound effects for each color
- Increasing difficulty as the sequence grows
- Game over feedback with sound and visual effects
- Restart functionality
